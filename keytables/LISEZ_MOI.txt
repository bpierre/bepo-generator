Disposition de clavier bépo, version 1.0
http://www.bepo.fr/


Description
-----------

Ce paquetage contient la disposition de clavier bépo, optimisée pour le français, créée par la communauté sur http://www.bepo.fr/.
Les fichiers de ce paquetage sont destinés à être utilisés avec loadkeys pour modifier la configuration de clavier en mode console sous open solaris.


Utilisation
-----------

La commande
  
  loadkeys bepo
  
bascule le clavier en clavier bepo. Vous pouvez copier le fichier bepo dans le répertoire /usr/share/lib/keytables/type_6/ pour le rendre accessible de n'importe quel répertoire avec la commande

  loadkeys bepo
  
La commande

  loadkeys france
  
permet de revenir à un clavier AZERTY.
