Disposition de clavier Dvorak français BÉPO, version 1.0rc2
http://www.clavier-dvorak.org/


Description
-----------

Ce paquetage contient la disposition de clavier Dvorak français BÉPO, optimisée pour le français, créée par la communauté sur http://www.clavier-dvorak.org/.
Les fichiers de ce paquetage sont destinés à être utilisés avec loadkeys pour modifier la configuration de clavier en mode console sous linux.
Deux versions sont disponibles, une iso-8859-15 (où tous les caractères Unicode du clavier Dvorak français BÉPO ont été supprimés), l'autre en UTF-8 (complète à part des touches mortes spéciales qui n'existent pas en mode console.


Utilisation
-----------

La commande
  
  loadkeys fr-dvorak-bepo.map           [ ou fr-dvorak-bepo-utf8.map ]
  
bascule le clavier en clavier Dvorak français BÉPO. Vous pouvez copier le fichier fr-dvorak-bepo.kmap dans le répertoire /usr/lib/kbd/keymaps/i386/dvorak/ pour le rendre accessible de n'importe quel répertoire avec la commande

  loadkeys fr-dvorak-bepo
  
La commande

  loadkeys fr
  
permet de revenir à un clavier AZERTY.
